# [CRUD with MongoDb and NodeJS](https://github.com/thulin82/crud-mongodb-nodejs)

Setup
---------------------
### Get project
```
$ git clone git@github.com:thulin82/crud-mongodb-nodejs.git
```
### Get dependencies
```
$ npm install
```
### Run app
```
$ npm start
```
